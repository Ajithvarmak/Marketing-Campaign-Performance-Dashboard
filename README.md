# Marketing Campaign Performance Dashboard

This project focuses on analyzing marketing campaign performance using Power BI.
The dashboard provides clear insights into revenue, spend, ROI, and conversions across different marketing channels.

## Project Objective
To visualize and evaluate marketing campaign performance and support data-driven marketing decisions.

## Tools Used
- Power BI Desktop
- Microsoft Excel

## Dataset
The dataset contains marketing campaign information such as campaign name, channel, spend, revenue, conversions, and dates.
Sample data is used for analysis and learning purposes.

## Dashboard Features
- KPI cards for Total Revenue, Total Spend, ROI, and Conversions
- Campaign-wise and channel-wise performance analysis
- Interactive charts and filters for better insights

## Files Included
- Power BI Dashboard (.pbix)
- Marketing Campaign Dataset (.xlsx)
- Project Documentation (PDF)

## Conclusion
This project demonstrates how Power BI can be used to transform raw marketing data into meaningful insights through interactive dashboards.
